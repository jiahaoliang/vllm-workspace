# kv_offload

`kv_offload` 是围绕 vLLM / vLLM Ascend / Mooncake 的跨仓特性开发目录。

## 涉及仓库

- `repos/vllm`: `kv_offload`
- `repos/vllm-ascend`: `kv_offload`
- `repos/Mooncake`: `main`，默认只读

## 资料

- `references/sources.md`: 外部资料链接索引。
- `references/snapshots/`: Markdown 快照。
- `notes/`: 本地设计和调查笔记。

## 状态

- `status.md`: 当前阶段和下一步。
- `repo-state.md`: 三个源码仓库的人类可读状态。
- `sync-log.md`: 上游和合作者同步记录。
